\begin{center}
{\Large \textbf{DNA foundation models encode motif identity but fail to learn regulatory grammar that exists}}
\end{center}

\section*{Background}

Whether enhancer activity is governed by motif arrangement or by more permissive logic remains a central question in cis-regulatory biology~[1,2].
We hypothesize that DNA foundation models learn \textbf{billboard-like representations}---encoding which transcription factor binding sites are present (absolute context) rather than their relative positions, orientations, or spacing (positional context).
This is distinct from asking whether biology itself follows billboard logic; we specifically test what models learn.
The prevailing view assumes foundation models learn positional syntax, but this has not been rigorously tested with proper controls for compositional confounds.
We derive testable predictions and evaluate them using a spacer-factored grammar sensitivity framework (SF-GSI) that isolates positional context from absolute context---resolving a critical confound in standard vocabulary-preserving shuffles, which perturb both simultaneously.
Crucially, we also test whether grammar matters biologically by identifying MPRA sequence pairs with identical motif vocabulary but different arrangements.

\section*{Methods}

SF-GSI computes a grammar sensitivity index, $GSI = \sigma(\hat{y}) / |\mu(\hat{y})|$, over vocabulary-preserving motif shuffles and factors the response into positional versus absolute context using matched-spacer perturbations and null-normalized testing.
The key prediction derived from our billboard model is concrete: if models encode only absolute context, then controlling for spacer composition should collapse apparent grammar sensitivity to noise---i.e., the residual positional signal should be statistically indistinguishable from a null distribution.

The framework proceeds in three stages.
First, a \textit{factorial decomposition} isolates the contributions of motif position, motif orientation, and spacer composition by generating matched perturbation sets in which each factor is varied independently while the others are held fixed; effect sizes are expressed as the fraction of full-shuffle variance attributable to each factor.
Second, a \textit{spacer ablation series} applies graded compositional perturbations---motif-only rearrangement, dinucleotide-preserving shuffle, GC-content shift, and full random replacement---to quantify the dose--response relationship between compositional disruption and model prediction change.
Third, a \textit{feature decomposition} regresses model predictions against explicit compositional features (GC content, dinucleotide frequencies, trinucleotide counts, and DNA shape parameters) to estimate how much of the expression signal is explained by absolute context alone.
Statistical significance is assessed via a three-stage correction cascade: an initial F-test on raw GSI, followed by z-score null normalization against matched shuffles, and finally Benjamini--Hochberg FDR correction at $\alpha = 0.05$.
Positive controls use synthetic constructs with fixed spacer sequence and known orientation-dependent activity, verifying that the framework retains power to detect genuine positional effects when they exist.

We applied SF-GSI to five MPRA datasets (Agarwal, Klein, Inoue, Vaishnav, and Jores), totaling 7,650 model--enhancer measurements: 3 foundation models $\times$ 5 datasets $\times$ 500 enhancers, plus Enformer on 3 human datasets $\times$ 50 enhancers.
The model set spans transformer, state-space, and MPRA-native architectures---DNABERT-2, Nucleotide Transformer v2, HyenaDNA, Enformer, and PARM~[3--8]---ensuring that conclusions are not artifacts of a single model family. Code: \url{https://github.com/bryanc5864/GRAMLANG}.

\section*{Results}

\textbf{Models learn billboard-like.} Under naive thresholding all tested enhancers appear grammar-sensitive, but this falls to 8.3\% after null normalization and to 0.12\% (9/7,650) after FDR correction (Fig.~\ref{fig:census}).
Per-enhancer classification shows 89.7\% (95\% CI: 87.7--91.7\%) of enhancers are ``billboard class'' with no detectable grammar effects.
Spacer composition accounts for 78--86\% of grammar-sensitivity variance (Fig.~\ref{fig:spacer}), while motif-only rearrangements produce 2--6$\times$ smaller effects than spacer perturbations.

\textbf{But grammar matters biologically.} We identified 363 MPRA sequence pairs with identical motif vocabulary but different arrangements.
These pairs show measured expression differences up to $\Delta = 6.3$ (Jores dataset), with 76\% having $|\Delta| > 0.5$.
This is direct experimental evidence that arrangement affects expression---grammar exists.

\textbf{Models fail to learn grammar.} We tested whether models can detect grammar when it demonstrably exists: (1) synthetic sequences with obvious grammar rules (motif order, spacing) yielded $p = 0.73$---not detected; (2) known helical periodicity (10.5bp, from BPNet literature) yielded $p = 0.96$---not detected.
Models cannot learn even simple, known grammar patterns.

Positive controls confirm genuine grammar is detectable with controlled spacers ($p \le 7.2\times10^{-15}$), but this requires experimental designs that current models were not trained on.

\section*{Conclusions}

These results reveal a critical limitation: DNA foundation models learn billboard-like representations---encoding what binding sites are present---but fail to learn positional grammar, even when it exists biologically.
We found 363 sequence pairs in MPRA data with identical motif vocabulary but different arrangements that show measured expression differences up to $\Delta = 6.3$, proving grammar matters.
Yet models cannot detect even synthetic grammar rules ($p = 0.73$) or known helical periodicity ($p = 0.96$).
The billboard finding (89.7\% of enhancers, 95\% CI: 87.7--91.7\%) reflects model limitations, not biological reality.
This reframes model evaluation: current architectures systematically miss regulatory grammar, and benchmarks should test whether models can learn grammar that demonstrably exists in experimental data.
